## Requirements

Operating system:

* Ubuntu/Debian
* CentOS/RHEL
* MacOS
* Windows

Node:

* NodeJS >= 10.x
* NPM >= 6.x

Database:

* MongoDB >= 3.6
* MySQL >= 5.6
* MariaDB >= 10.1
* PostgreSQL >= 10

### Installation

```bash
npm i && npm run install
```

### Quick start

```bash
cd server && npm run dev
```

```bash
cd client && npm run dev
```
